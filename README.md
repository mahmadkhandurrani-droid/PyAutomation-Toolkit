# PyAutomation-Toolkit

A modular Python automation system for file backup, organization, renaming, and reporting. Built to simulate real-world office workflows with logging, CLI interface, and config-driven architecture.

---

## 📌 Overview

PyAutomation-Toolkit is designed to automate repetitive file management tasks such as:

- Secure file backups with timestamped folders
- Automatic folder organization by file type
- Batch file renaming system
- Activity logging for all operations
- Simple command-line interface (CLI)
- Config-based customization

This project demonstrates practical Python skills used in real automation environments.

---

## ⚙️ Features

- 📁 **File Backup System**
  - Copies files safely using `shutil.copy2`
  - Creates timestamped backup folders
  - Prevents data overwrite

- 🗂 **File Organizer**
  - Automatically sorts files by extension
  - Creates structured folders (PDF, JPG, TXT, etc.)

- ✏️ **Batch Renamer**
  - Renames multiple files in sequence
  - Ensures consistent naming structure

- 📊 **Logging System**
  - Tracks all operations with timestamps
  - Logs errors and successful actions
  - Helps debugging and audit tracking

- 🧾 **Report Generator**
  - Generates summary reports of operations performed

- 🖥 **CLI Interface**
  - Menu-driven system for easy user interaction

- ⚙️ **Config Driven**
  - Uses `config.json` for folder paths and settings
